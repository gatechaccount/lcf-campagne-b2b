# Campagne réactivation Pro · La Clôture Française

Mini-site statique présentant la campagne de réactivation B2B (été 2026), menée par Growth Angels pour La Clôture Française.

## Contenu

| Fichier | Rôle |
| --- | --- |
| `index.html` | Page d'accueil reliant les trois vues |
| `sequence-revendeurs.html` | Séquence de 4 e-mails · cible revendeurs (1 122 contacts) |
| `sequence-artisans.html` | Séquence de 4 e-mails · cible artisans (31 contacts) |
| `formulaire-artisans.html` | Formulaire de qualification artisans (10 questions) |

Les liens « Mettre à jour ma fiche » de la séquence artisans pointent vers `formulaire-artisans.html`.

## Publication sur GitHub Pages

1. Créer un dépôt GitHub et y déposer ces fichiers (racine).
2. Dans **Settings → Pages**, choisir la branche `main` et le dossier `/ (root)`.
3. La page d'accueil sera servie à l'adresse fournie par GitHub Pages.

## À finaliser

- Brancher l'envoi du formulaire (Airtable ou HubSpot) : voir le commentaire dans la balise `<script>` de `formulaire-artisans.html`.
- Renseigner les liens de prise de rendez-vous (`href="#"`) et les champs entre crochets `[ ]` dans les e-mails.
- Compléter les trois modèles/gammes qui performent dans le mail 2 de la séquence revendeurs.
